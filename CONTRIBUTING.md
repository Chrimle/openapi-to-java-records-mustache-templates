# Contributing

> [!IMPORTANT]
> Contributing, in terms of feedback - as feature requests or bug reports are very much appreciated!
> 
> Contributing, _in terms of opening PRs and/or committing changes_, are **blocked**.

## Channels

There are multiple ways to reach out and/or discuss topics.

### [Issues](https://github.com/Chrimle/openapi-to-java-records-mustache-templates/issues)
If relevant for a particular GitHub Issue, whether that is a feature or bug, is preferably discussed in the issue itself.

### [Discussions](https://github.com/Chrimle/openapi-to-java-records-mustache-templates/discussions)
If it is a broader question and/or discussion.
